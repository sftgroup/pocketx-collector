⚡ Quickstart: Allocate a User Address
Copy-paste this into your terminal. Uses your production API key. Returns a unique deposit address for this user.
# Step 1: Create a deposit address for a user curl "https://api.pocketx.app/v2/saas/address" \ -H "x-api-key: pk_live_8a3f..." \ -H "Content-Type: application/json" \ -d '{"user_id": "player_9999", "chain": "sepolia"}' # Response: # {"code":0,"data":{"address":"0x3f2a...","user_id":"player_9999","chain":"sepolia"}} # Step 2: Check user balance curl "https://api.pocketx.app/v2/saas/address/player_9999" \ -H "x-api-key: pk_live_8a3f..." # Step 3: Webhook will POST to your URL when deposit arrives # POST https://your-app.com/webhook {"event":"deposit","user_id":"..."}
